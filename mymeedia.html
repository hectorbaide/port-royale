work about

My Meedia

Design explorations for an integrated media management system

Project Type

Interface design

Project Duration

6 months

Role

Interface designer

Mymeedia is a content management, search and distribution service. The core idea was to be able to explore collections from different sources in an integrated environment, mixing different types of media and types of information into a unified stream.

Promotional video created to explain MyMeedia


I created a system for displaying the information by itself or integrated within a website. It was based on the idea of combining "content cards" within pages to display things in a modular way.



In addition to the concept of this interface system I developed some examples which used the principle in different context for research and promotional reasons.


Example for a fashion aggregator


Example in a startup accelerator website

Example in a portfolio website

The project was done within IN2 Search Interfaces and was used for generating leads and different business opportunities for the Mymeedia product.
